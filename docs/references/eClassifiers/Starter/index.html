* An event-source (starter) activity that triggers the process. Corresponds to pd:starter in the process XML. Examples: HTTP Event Source, JMS Queue Receiver, Timer, File Poller.

Structurally identical to a regular Activity; modelled as a distinct class so that tooling can easily distinguish starters from regular steps.
